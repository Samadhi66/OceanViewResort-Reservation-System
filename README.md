# Ocean View Resort Reservation System

This project is a web-based reservation management system developed using Java, JSP, Servlets, and MySQL.  

The system allows administrators and receptionists to manage hotel reservations, generate bills, and view reservation details.

## Technologies Used
- Java
- JSP / Servlets
- MySQL
- Bootstrap
- Git & GitHub

## Features
- User login authentication
- Add / edit / delete reservations
- Generate billing invoices
- Export reservation data
- Role-based access control
